# 各领域输出模板

根据领域选择对应模板，所有输出使用英文。

---

## Template A: photo_portrait (摄影-人像)

```
[风格名称] Portrait Photography Style

I. Basic Settings
1. Domain: photo_portrait
2. Style Direction: [cinematic/vintage/editorial/minimalist/etc.]
3. Aspect Ratio: [用户指定尺寸]

II. Style Core - Keep Unchanged
1. Equipment Simulation: [shot on digital mirrorless camera / 35mm film camera / medium format]
2. Technical Parameters:
   - Aperture: [f/1.4 - f/2.8] creating [shallow depth of field with creamy bokeh]
   - Focal Length: [85mm / 50mm / 35mm] [portrait compression / natural perspective]
   - Focus: [sharp on eyes / soft overall]
3. Lighting Setup:
   - Light Type: [golden hour sunlight / studio strobe / window light / overcast]
   - Light Position: [Rembrandt / butterfly / split / rim / three-point]
   - Light Quality: [soft diffused / hard directional / mixed]
   - Color Temperature: [warm 3200K / neutral 5500K / cool 6500K]
4. Color Tone: [desaturated film look / vibrant editorial / moody low-key / bright high-key]
5. Composition: [rule of thirds / centered / negative space / environmental]
6. Post-processing:
   - Saturation: [muted / natural / enhanced]
   - Contrast: [low / medium / high]
   - Color Grading: [warm shadows cold highlights / teal and orange / monochromatic]
   - Effects: [film grain / vignette / faded blacks]
7. Skin Rendering: [natural texture / slightly smoothed / porcelain / sun-kissed]

III. Customizable Elements
1. Subject: {gender}, {age range}, {ethnicity}, {hair style and color}
2. Outfit: {clothing type}, {clothing color and material}, {accessories}
3. Expression & Pose: {facial expression}, {body pose}, {hand placement}
4. Setting: {location type}, {specific environment}, {time of day}
5. Background: {background elements}, {props}, {depth}

IV. Mood & Atmosphere
[emotional keywords: intimate, powerful, vulnerable, confident, serene, etc.]

Negative Prompts: [over-processed skin, unnatural colors, harsh shadows, busy background, etc.]
```

## Template C: painting_realistic (绘画-写实)

```
[风格名称] Realistic Painting Style

I. Basic Settings
1. Domain: painting_realistic
2. Style Direction: [oil painting / digital painting / concept art / hyperrealism]
3. Aspect Ratio: [用户指定尺寸]

II. Style Core - Keep Unchanged
1. Art Style: [classical oil / impressionistic / photorealistic digital / concept art]
2. Stroke Characteristics:
   - Technique: [glazing thin transparent layers / impasto thick textured / alla prima wet-on-wet / sfumato smoke-like blending / dry brush scumbling / broken color short strokes / fine blending invisible strokes / palette knife flat scraping / calligraphic one-stroke forming]
   - Paint Thickness: [extremely thin and transparent / thin / medium / thick / heavily built-up impasto]
   - Visibility: [invisible fully blended / subtly visible / clearly visible / prominently emphasized]
   - Edge Quality: [sharp hard edges / clear but soft / soft gradual blending / blurred dissolving / broken rough]
   - Rhythm: [even and uniform / following form contours / free and loose / rapid and energetic / slow and delicate]
   - Direction: [following form / expressive / uniform / radiating / cross-hatched]
3. Rendering Quality: [highly detailed with subtle surface texture / painterly with visible artistic hand / loose realistic suggesting form / photorealistic with no visible strokes]
4. Color Palette:
   - Dominant Colors: [specific color descriptions, e.g. "deep aquamarine blue-green" not just "blue"]
   - Color Relationship: [analogous / complementary / triadic / split-complementary / monochromatic]
   - Saturation Distribution: [uniform across painting / high saturation on subject low on background / selective saturation on focal points / overall muted grey tones]
   - Temperature Distribution: [uniformly warm / uniformly cool / warm-cool interplay with warm on {areas} and cool on {areas} / gradual temperature shift from {area} to {area}]
   - Color Transparency: [opaque and dense / semi-transparent layered glazing / luminous glowing from within / murky and diffused]
5. Light & Shadow:
   - Light Source: [natural sunlight / moonlight / ambient scattered / artificial / penetrating through medium / self-luminous / multiple sources]
   - Light Direction: [top / side / backlit / bottom / wrap-around / scattered no direction]
   - Light Quality: [hard with sharp edges / soft with gradual falloff / scattered ambient / dappled through obstructions / caustic patterns through water or glass]
   - Value Contrast: [extreme chiaroscuro / high / moderate / low even illumination / very low high-key or low-key]
   - Shadow Treatment: [opaque black / dark but colored with {cold/warm} bias / transparent shadows with visible detail / reflected fill light]
   - Light-Shadow Color: [warm light cool shadow / cool light warm shadow / unified temperature / colored light with {color} / environment color reflected]
6. Material Rendering (describe ALL major materials visible in the image):
   - Skin: [dry matte / wet glistening / underwater with subsurface glow / sweating with highlights / makeup with {finish}] — note subsurface scattering level, pore visibility, color transitions
   - Hair: [dry with {luster type} / wet clinging / floating in water / windswept / braided] — note individual strand visibility, translucency, volume rendering
   - Fabric: [{material} with {drape quality}] — note fold rendering, translucency, weight
   - Water/Liquid: [{state: still surface / flowing / underwater / droplets / splashing}] — note transparency, refraction distortion, depth color attenuation, bubbles
   - Other Surfaces: [list each significant material with its rendering approach]
   (Only include materials actually present in the image. Omit categories not visible.)
7. Environment Medium (include ONLY if a non-air medium is present):
   - Medium Type: [clear water / murky water / thin fog / dense fog / smoke / rain / snow / dust]
   - Visibility Falloff: [sharp nearby blurring with distance / uniform low visibility / layered attenuation]
   - Color Shift: [blue-green shift with red absorption underwater / white-grey washout in fog / warm haze]
   - Light Scattering: [god rays / Tyndall effect / uniform diffusion / caustic patterns on surfaces]
   - Edge Softening: [contour blurring / color bleeding / halation glow]
   - Suspended Particles: [bubbles / dust motes / pollen / snowflakes / water droplets / light bokeh]
8. Element Interactions (describe visible cross-element effects):
   - Light-Medium-Surface: [caustic light patterns on skin through water / dappled shadows through foliage / colored light through stained glass]
   - Material-Medium: [hair spreading and floating in water / fabric becoming translucent when wet / metal losing luster in fog]
   - Reflection-Environment: [water surface color reflecting on face / green leaf bounce light on skin / metallic environment mapping]
   - Edge Interactions: [soft blending of skin into water / sharp hair strands against blurred background / foreground dissolving into fog]
   (Only include interactions actually visible in the image.)
9. Perspective: [one-point / two-point / atmospheric perspective / isometric / fish-eye distortion]

III. Customizable Elements
1. Character: {gender}, {age}, {species or race}, {body type}
2. Design: {outfit design}, {hairstyle}, {accessories}, {color scheme}
3. Expression & Pose: {emotion}, {pose}, {gesture}, {eye direction}
4. Scene: {environment type}, {props}, {narrative elements}
5. Theme: {story context}, {symbolic elements}

IV. Mood & Atmosphere
[artistic keywords: epic, melancholic, heroic, mysterious, romantic, etc.]

Negative Prompts: [flat rendering, inconsistent lighting, anatomical errors, muddy colors, etc.]
```

## 使用说明

1. 根据领域识别结果选择对应模板
2. `II. Style Core` 部分使用完整英文句子描述
3. `III. Customizable Elements` 部分用 `{}` 包裹可替换内容
4. 所有描述应具体、可操作，避免模糊词汇
5. 负面提示词针对该领域常见的风格偏差
