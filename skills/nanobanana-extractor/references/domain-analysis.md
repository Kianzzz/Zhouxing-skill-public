### 笔触技法细分表

分析绘画类图片时，必须从以下维度精确判断，**禁止笼统使用"厚涂"概括所有油画风格**：

| 技法 | 英文术语 | 视觉特征 | 常见误判 |
|------|----------|----------|----------|
| 古典罩染 | Glazing | 薄透明层叠加，色彩通透有深度，表面光滑 | 容易误判为"厚涂" |
| 厚涂/堆叠 | Impasto | 颜料堆积明显，可见立体笔触纹理，有浮雕感 | 这才是真正的厚涂 |
| 直接画法 | Alla prima | 湿上加湿，一次完成，色彩鲜活但边缘柔和 | 容易误判为"厚涂" |
| 晕染/渐融 | Sfumato | 极度柔和过渡，无可见笔触边界，如烟雾般 | 容易误判为"数字绘画" |
| 干笔/擦扫 | Dry brush / Scumbling | 干涩笔触，底层隐约可见，纹理粗糙 | 容易误判为"粗犷厚涂" |
| 点彩 | Pointillism | 小点状笔触并置，远看混色 | — |
| 印象派短笔触 | Broken color | 短促可见笔触，色彩在视网膜混合 | 容易误判为"厚涂" |
| 细腻写实 | Fine blending | 笔触不可见或极细微，追求照片般平滑 | 容易误判为"数字绘画" |
| 书写性笔触 | Calligraphic strokes | 一笔成形，有起收笔变化，类似书法 | — |
| 刮刀/画刀 | Palette knife | 宽平扁刮痕，边缘锐利，颜料层平整但厚 | 与传统厚涂不同 |

### 笔触质感维度

对笔触的描述必须覆盖以下 4 个独立维度，每个维度独立判断：

| 维度 | 选项范围 | 说明 |
|------|----------|------|
| **颜料厚度** | 极薄透明 / 薄 / 中等 / 厚 / 极厚堆积 | 颜料在画布上的物理厚度 |
| **笔触可见度** | 不可见(完全融合) / 隐约可见 / 清晰可见 / 突出强调 | 能否看到单独的笔触痕迹 |
| **边缘质感** | 锐利硬边 / 清晰但柔和 / 柔和渐融 / 模糊消散 / 破碎毛糙 | 笔触与周围区域的交界处理 |
| **笔触节奏** | 均匀规整 / 跟随形体 / 自由松散 / 急促有力 / 缓慢细腻 | 笔触的运动感和速度感 |

> **使用示例**：
> - 莫奈风格 → 颜料中等厚度、笔触清晰可见、边缘破碎毛糙、笔触急促有力
> - 布格罗风格 → 颜料薄至中等、笔触不可见、边缘柔和渐融、笔触缓慢细腻
> - 梵高风格 → 颜料极厚堆积、笔触突出强调、边缘清晰但柔和、笔触急促有力

### 配色深度分析

分析配色时必须覆盖以下 5 个子维度，不可只写"暖色调"或"冷色调"了事：

| 子维度 | 选项范围 | 说明 |
|--------|----------|------|
| **主色调** | 具体描述（如 "深海蓝绿" 而非 "蓝色"） | 画面中占比最大的色彩 |
| **配色关系** | 邻近色/互补色/三角色/分裂互补/单色系 | 色彩之间的色相关系 |
| **饱和度层次** | 全画面统一 / 主体高饱和背景低饱和 / 选择性饱和 / 整体灰调 | 饱和度如何分配 |
| **色彩温度分布** | 全暖 / 全冷 / 冷暖交织(说明哪里暖哪里冷) / 随深度渐变 | 画面不同区域的色温关系 |
| **色彩透明度** | 不透明厚重 / 半透明层叠 / 通透发光(glazing效果) / 混浊 | 颜料层的光学效果 |

### 光影深度分析

光影分析必须覆盖以下 6 个子维度：

| 子维度 | 选项范围 | 说明 |
|--------|----------|------|
| **光源类型** | 自然光(日光/月光) / 人造光 / 环境散射光 / 自发光 / 穿透光(透过介质) / 多光源 | 光从哪来 |
| **光线方向** | 顶光/侧光/逆光/底光/环绕光/散射无方向 | 光照射的角度 |
| **光线质感** | 硬光(清晰边界) / 柔光(渐变边界) / 散射光(无明确方向) / 斑驳光(透过遮挡物) / 焦散光(caustics，透过水或玻璃产生的光纹) | 光的硬度和形态 |
| **明暗对比度** | 极高(明暗分明chiaroscuro) / 高 / 中等 / 低(整体均匀) / 极低(高调或低调) | 最亮与最暗的差距 |
| **暗部处理** | 纯黑不透 / 暗但有色彩(冷暗/暖暗) / 透明暗部(可见细节) / 反射补光 | 阴影区域的画法 |
| **光色关系** | 暖光冷影 / 冷光暖影 / 统一色温 / 彩色光(说明颜色) / 环境色反射 | 光与影的色彩关系 |

### 材质渲染细分表

材质表现的必要性提升为 ★★★。分析时必须识别画面中出现的**所有主要材质**，并描述其渲染技法：

| 材质类别 | 细分材质 | 渲染关注点 |
|----------|----------|------------|
| **皮肤** | 干燥/湿润/水下/出汗/带妆 | 次表面散射(subsurface scattering)程度、毛孔可见度、光泽度、色彩冷暖过渡 |
| **毛发** | 干发/湿发/水中飘散/风中飘动/编织 | 发丝可见度、光泽类型(丝绸光/哑光)、透光度、体积感表现方式 |
| **织物** | 丝绸/棉麻/皮革/蕾丝/纱/厚重织物 | 褶皱渲染方式、透光性、表面纹理、重力感 |
| **水/液体** | 静水面/流动水/水下/水滴/飞溅/涟漪 | 透明度、折射扭曲、表面反射、深度色彩衰减、气泡 |
| **金属** | 光滑镜面/拉丝/锈蚀/古旧氧化 | 反射清晰度、环境色映射、高光形状 |
| **玻璃/透明体** | 清玻璃/磨砂/有色/冰 | 折射失真度、边缘高光、内部色散 |
| **有机物** | 花瓣/叶片/木纹/石头/泥土 | 表面微纹理、含水量表现、半透明度(如阳光下的叶片) |
| **特效材质** | 烟/火/魔法光/星空/云 | 发光方式、边缘消散方式、与环境的融合度 |

> **关键提醒**：如果图片中某种材质的渲染方式是风格的核心特征（如水下场景中水的渲染），
> 则该材质必须作为**共性元素**详细描述，不能归入个性化元素。

### 环境介质与视觉影响

当画面存在非空气介质（水下、雾中、烟尘、雨中等）或特殊大气条件时，**必须分析该介质对所有视觉元素的影响**：

| 子维度 | 说明 | 示例 |
|--------|------|------|
| **介质类型** | 画面中的传播介质是什么 | 清水/浑浊水/薄雾/浓雾/烟/雨/雪/沙尘 |
| **可见度衰减** | 随距离增加，细节和对比度如何衰减 | 近处清晰远处模糊/均匀低可见度/分层衰减 |
| **色彩偏移** | 介质对色彩的吸收和改变 | 水下偏蓝绿且红色衰减/雾中偏白灰/黄昏雾偏暖 |
| **光线散射** | 介质中光如何传播 | 丁达尔效应(god rays)/均匀散射/焦散纹(caustics) |
| **边缘软化** | 介质对物体边缘的影响 | 轮廓柔化/色彩溢出/光晕 |
| **悬浮粒子** | 介质中的微粒 | 气泡/灰尘/花粉/雪花/水滴/光斑 |

> **如果画面中存在环境介质，这通常是风格的决定性特征，必须作为高优先级共性元素。**

### 元素交互渲染

描述不同元素之间的视觉交互效果——这往往是区分精细风格与粗糙风格的关键：

| 交互类型 | 说明 | 示例 |
|----------|------|------|
| **光-介质-表面** | 光穿过介质照射到表面的效果 | 水下焦散光纹映在皮肤上、透过树叶的斑驳光影、透过玻璃的彩色光斑 |
| **材质-介质** | 物体在介质中的视觉变化 | 头发在水中飘散展开、织物在水中变透明并贴合身体、金属在雾中失去光泽 |
| **反射-环境** | 表面对周围环境的色彩反射 | 水面环境色反射在脸上、绿叶反射光打在皮肤上呈绿色、金属反映环境 |
| **边缘交互** | 不同元素交界处的渲染方式 | 皮肤与水的交界(柔和/清晰)、头发丝与背景的融合方式、前景与雾的消融 |

## 6. general (通用)

### 分析策略

1. 首先尝试识别最接近的已知领域特征
2. 创建混合分析框架
3. 重点关注：色彩、构图、整体氛围
4. 共性/个性化区分原则不变

### 通用检查清单

| 维度 | 说明 |
|------|------|
| 整体风格 | 能否用已有术语描述？ |
| 色彩特征 | 主色调、配色规律、饱和度 |
| 构图特征 | 元素排列、视觉焦点、空间关系 |
| 质感纹理 | 光滑/粗糙/噪点/颗粒 |
| 氛围情绪 | 主要传达的情感 |

## 质量检查通用标准

### 必检项目（所有领域）

- [ ] 是否使用自然语言完整句子？
- [ ] 共性元素是否足够具体？
- [ ] 个性化元素是否用 `{}` 正确标注？
- [ ] 是否生成了负面提示词？
- [ ] 替换 `{}` 后风格能否保持一致？

### 评分门槛

- 共性元素复用评分 ≥ 7 的数量应 ≥ 60%
- 如不达标，需补充细化共性元素描述
