# 微信文章提取实用指南

## ⚠️ 重要说明

经过实际测试，微信的反爬虫机制非常严格：
- ❌ **海外IP直接拦截**：Firecrawl、WebFetch等海外服务会触发安全验证
- ❌ **无Cookie访问被限制**：直接HTTP请求也会被拦截
- ❌ **Playwright 网络超时**：访问微信服务器很慢

## ✅ 实际可行的方案

### 方案1：批量搜索模式（最推荐）⭐⭐⭐

**这是目前最可靠的方式**，使用微信官方公众号后台 API：

#### 优点
- ✅ **不触发安全验证**：使用官方API
- ✅ **稳定可靠**：直接从微信后台获取
- ✅ **功能强大**：可获取所有历史文章、搜索、筛选
- ✅ **速度快**：官方接口，无需爬虫

#### 使用步骤

**首次使用（需要5分钟设置）：**

```
您：搜索"婷在路上的日子"这个公众号

Claude：
检测到首次使用，需要登录...

1. 准备工作：
   - 需要一个微信公众号（个人订阅号，免费）
   - 如果没有，去 https://mp.weixin.qq.com/ 注册

2. 开始登录：
   - 生成二维码
   - 用微信扫码，选择你的公众号
   - 登录成功（token 有效期3个月）

3. 开始搜索：
   ✅ 找到公众号，显示所有历史文章
   ✅ 选择需要的文章下载
```

**后续使用（无需再次登录）：**

```
您：搜索"洞见"这个公众号

Claude：
检测到已登录，直接搜索...
✅ 找到 1 个公众号
✅ 使用缓存数据（共 500 篇文章）
📊 显示文章列表...
```

#### 适用场景
- ✅ 需要系统性整理某个公众号的内容
- ✅ 需要批量下载多篇文章
- ✅ 需要搜索和筛选文章
- ✅ 对稳定性要求高

### 方案2：使用 Cookie（单篇文章）⭐⭐

如果只需要提取单篇文章，可以从浏览器复制 Cookie：

#### 步骤

**1. 在浏览器打开文章**
   - 在 Chrome/Safari 中打开微信文章
   - 确保文章能正常显示

**2. 打开开发者工具**
   - macOS: `Cmd + Option + I`
   - Windows: `F12`

**3. 复制 Cookie**
   - 切换到 **Network** 标签
   - 刷新页面（`Cmd + R`）
   - 点击第一个请求（通常是文档请求）
   - 找到 **Request Headers** 部分
   - 复制 **Cookie** 字段的值

**4. 使用 Cookie 提取**

```bash
python3 scripts/extract_article.py \
  "https://mp.weixin.qq.com/s/xxxxx" \
  ./output \
  --cookie "your_cookie_string_here"
```

**或者在 Claude 中：**

```
您：帮我提取这篇文章，使用以下 Cookie：
   [粘贴 Cookie 字符串]
   https://mp.weixin.qq.com/s/xxxxx

Claude：
使用提供的 Cookie 提取...
✅ HTTP请求成功！
已保存到：2026-01-11_文章标题.md
```

#### 优点
- ✅ 可以绕过安全验证
- ✅ 适合单篇或少量文章
- ✅ 无需公众号登录

#### 缺点
- ❌ Cookie 会过期（通常几小时）
- ❌ 每次需要手动复制
- ❌ 不适合批量操作

### 方案3：本地网络直连（有限场景）⭐

**前提条件**：
- 您的电脑在**国内**
- Shadowrocket 的直连规则已生效

#### 验证直连是否生效

```bash
# 测试访问速度
time curl -I https://mp.weixin.qq.com

# 如果返回很快（<2秒），说明直连生效
# 如果超时或很慢，说明仍走代理
```

#### 如果直连生效

可以直接使用 HTTP 请求方式：

```bash
python3 scripts/extract_article.py "https://mp.weixin.qq.com/s/xxxxx"
```

#### 如果直连不生效

检查 Shadowrocket 配置：
1. 打开 Shadowrocket
2. 配置 → 编辑配置
3. 确认有以下规则：
   ```
   DOMAIN-SUFFIX,mp.weixin.qq.com,DIRECT
   DOMAIN-SUFFIX,weixin.qq.com,DIRECT
   ```
4. 规则要在**最上面**（优先匹配）

## ❓ 常见问题

### Q1: 为什么 Firecrawl 会失败？
**A**: Firecrawl 使用海外服务器，会被微信识别并拦截。微信的反爬虫机制会检测IP地址来源。

### Q2: 批量搜索模式需要什么？
**A**: 需要一个微信公众号（个人订阅号，免费）。注册大约5分钟，之后可以长期使用。

### Q3: Cookie 从哪里获取？
**A**:
1. 浏览器打开微信文章
2. F12 打开开发者工具
3. Network 标签 → 刷新页面
4. 点击第一个请求 → 复制 Cookie

### Q4: Cookie 会过期吗？
**A**: 会的，通常几小时后过期。过期后需要重新获取。

### Q5: 直接提取为什么都失败？
**A**: 微信的反爬虫机制很严格：
- 海外IP直接拦截
- 无Cookie访问被限制
- 检测User-Agent
- 检测访问频率

### Q6: Shadowrocket 配置了直连，为什么还是失败？
**A**: 可能原因：
1. 规则顺序不对（需要在最上面）
2. Playwright 等工具没有继承系统代理
3. 即使走直连，无Cookie仍会被拦截

### Q7: 哪种方案最可靠？
**A**: **批量搜索模式**（方案1）最可靠：
- 使用微信官方API
- 不触发安全验证
- 稳定快速
- 功能强大
