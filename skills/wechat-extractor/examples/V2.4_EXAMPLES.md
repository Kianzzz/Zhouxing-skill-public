## 🎯 最佳实践

### 1. 让 Claude 自动选择

✅ **推荐**：
```
用户: 提取这篇文章 https://mp.weixin.qq.com/s/xxxxx
```

❌ **不推荐**：
```
用户: 用 Playwright 提取这篇文章
```

**原因**：让 Claude 自动选择，会优先使用更快更稳定的方式。

### 2. 批量提取时设置延迟

当批量提取多篇文章时，建议添加延迟：

```
用户: 帮我提取这 10 篇文章，每篇间隔 5 秒
```

### 3. 遇到问题时查看日志

每种方式都会输出详细日志，帮助定位问题：

```
⏳ 尝试 HTTP 请求...
❌ HTTP 请求失败：请求超时（网络可能较慢）
   → 这说明网络连接有问题

⏳ 尝试 WebFetch...
✅ WebFetch 成功！
   → 改用 WebFetch 成功解决
```

## 🔍 故障排除示例

### 场景 1: 所有方式都失败

```
❌ 提取失败：所有提取方式都失败了

可能原因：
1. 文章链接无效或已被删除
2. 文章需要关注公众号后才能查看
3. 遇到严重的反爬虫限制

解决方法：
1. 在微信客户端打开文章，确认链接有效
2. 复制新的链接重试
3. 联系公众号作者获取文章内容
```

### 场景 2: 提取内容为空

```
✅ HTTP请求 成功！
❌ 保存失败：提取的内容为空

可能原因：
1. 文章主要是图片/视频，文字内容很少
2. 文章使用特殊格式，无法识别

解决方法：
1. 检查原文是否有文字内容
2. 如果是图片为主的文章，考虑手动复制
```
