# 五层解构分析框架

五层分析的详细指南，包含分析维度、抽象化规则、文件生成规范。

---

## 一、核心原则

- **描述性**：分析「是什么」而非「应该怎样」
- **去案例化**：严禁引用原文，所有具体内容抽象为【变量】形式
- **可复用性**：足够抽象（可迁移）又足够具体（可执行）
- **风格复制**：目标是帮用户「复制」风格，而非「改进」风格

### 抽象化规则

| 类型 | 错误 | 正确 |
|------|------|------|
| 人名 | "hi，我是Ray。" | 【问候语 + 作者名】格式 |
| 特定词汇 | "干、搞、拼、卷" | 【高能动词类型】 |
| 具体短语 | "说白了、算了" | 【总结性口语词类型】 |
| 具体比喻 | "供电系统、低电量" | 【系统隐喻类型：电子/机械/商业】 |
| 具体案例 | "眼神散了，皮肤灰了" | 【感官描述类型：视觉+触觉+动作】 |

## 二、五层分析详解

### 2.1 韵律节奏层

**目标**：识别并描述节奏特征，帮助再现。

#### 停顿密度计算

统计以下指标：
- 句号数（真实句子数）、逗号/分号/破折号数、总字数、段落数（按空行分隔）
- 句号密度 = 句号数 / 总字数 × 1000
- 逗号密度 = 逗号数 / 总字数 × 1000

#### 节奏模式识别（核心 - 防短句误判）

```
⚠️ 强制规则：
1. 句子边界 = 句号（。）/ 问号（？）/ 感叹号（！）
2. 段落边界 = 空行分隔（仅换行不空行 = 同一段落）
3. 换行 ≠ 句子结束
4. 平均句长 = 总字数 / 句号数
5. 段落数 >> 句号数 → "一句一段"排版，句子本身可能是中长句
```

三种模式：

| 模式 | 特征 | 平均句长 | 判断条件 |
|------|------|---------|---------|
| A: 短句顿挫型 | 句号密集 | < 15字 | 句号密度 ≥ 60/千字 |
| B: 长句内部顿挫型 | 句号稀疏，逗号密集 | 25-50字 | 句号密度 < 40/千字，逗号密度 ≥ 80/千字 |
| C: 视觉顿挫型 | 一句一段排版 | > 20字 | 平均句长 > 20字 且 段落数 > 句号数×1.5 |

阈值为经验参考值，可微调±10。

#### 其他维度

- **显性**：句长分布、节奏型（顿挫/流畅/交替）、停顿规律、标点节奏感
- **隐性**：基于其他4层推断的节奏特征
- **朗读感**：可朗读性、口语节奏感、听觉记忆点
- **音韵**：是否避免拗口、声韵搭配

**禁止**提供优化建议 | **允许**推断隐性规律

### 2.2 微观语言层

分析维度：
- **词汇颗粒度**：抽象vs具体、动词vs形容词、口语vs书面、术语密度
- **句式结构**：长/中/短句比例、主动/被动、陈述/疑问
- **修辞手法**：隐喻类型、排比、反问、留白
- **标点节奏**：各标点使用频率和位置规律
- **风格禁忌**：该风格应避免的表达
- **非功能性口语碎片**（反AI关键维度）：该作者文章中不服务论证、纯粹传递人格感的口语表达。重点提取：
  - 碎片类型：插入式补充（"对，就是XXX"）/ 自言自语（"算了不说了"）/ 信口评价（"他不吃那一套"）
  - 插入位置：句中、句末、独立成句
  - 典型示例：抽象为【口语碎片类型】形式记录

输出仅用"句式结构 + 语言调性描述"，严禁引用原文。

---

### 2.3 宏观结构层

分析维度：
- **整体框架**：总分总/递进式/并列式/故事线/问答式
- **段落设计**：段落长度分布、过渡方式、小标题规律
- **开头策略**：破题方式（疑问/对比/场景/数据/金句）
- **结尾策略**：收束方式（升华/行动号召/留白/呼应）
- **领域特有元素**：根据领域动态生成2-4个
- **结构不对称模式**（反AI关键维度）：该作者各段/各节的长度差异模式。重点提取：
  - 最长节与最短节的比例（如 3:1、2:1）
  - 是否有"展开区"（作者特别想多说的地方）和"略过区"（一笔带过的地方）
  - 段落数量的不规则性（某节5段，某节只有2段）
- **功能性冗余模式**（反AI关键维度）：该作者在文章中插入的非论证性文字类型。重点提取：
  - 冗余类型偏好：补丁式免责 / 括号式补充 / 自嘲式撤退 / 读者预设回应 / 事后追加 / 条件限定
  - 冗余频率：每篇约几处
  - 冗余位置偏好：段首/段中/段尾/文末
  - 典型示例：抽象为【冗余类型】形式记录

标注【恒定】和【适应】特征。严禁出现原文自我介绍格式或具体金句。

---

### 2.4 传播规律层

分析维度：
- **钩子设计**：类型（冲突/悬念/利益/身份认同/稀缺性）、分布、密度
- **情绪曲线**：起伏节奏、高潮点设置
- **情绪断点模式**（反AI关键维度）：该作者在情绪弧线中制造跳跃/中断的方式。重点提取：
  - 断点类型偏好：粗口/俚语突变 / 自嘲中断 / 幽默插入 / 冷静陈述打断煽情 / 轻描淡写收束
  - 断点频率：每篇约几处
  - 典型示例：抽象为【情绪断点类型】形式记录
- **代入感制造**：目标用户筛选、"这说的就是我"感
- **转发动机**：人设塑造、社交货币
- **信息差设计**：不对称位置
- **预期管理**：阅读预期设定
- **阅读流畅性**：降低障碍、"自然下滑"惯性

定性描述为主。

---

### 2.5 文化认知层

分析维度：
- **典故使用**：类型、频率、方式（直引/化用/暗喻）
- **知识领域**：涉及学科、密度、展示方式
- **术语策略**：密度、处理方式、功能
- **受众门槛**：大众/小众/专业
- **跨文化元素**：中西方比例、传统/现代比例
- **领域特有文化元素**：动态生成

## 三、共性提取规则（≥3篇时）

### 特征分类

| 标注 | 频率 | 处理 |
|------|------|------|
| 【恒定】 | ≥ 80% | 保留 |
| 【高频】 | 60-79% | 保留 |
| 【适应】 | 变化的 | 标注调整条件 |
| 【个性】 | < 40% | 剔除 |

### 区分领域 vs 个人特征

判断方法：同领域其他作者是否也这样做？
- 是 → 【领域特征】
- 否 → 【个人特征】（风格核心）

### 风格稳定性

| 级别 | 占比 | 说明 |
|------|------|------|
| 高 | 90%+ | 风格成熟，可直接复用 |
| 中 | 70-89% | 基本稳定，注意适应性 |
| 低 | <70% | 建议增加样本量 |

## 四、文件生成规范

### 通用文件结构

```markdown
# {文件名称}

> 领域：{领域}
> 样本数量：{数量}
> 生成时间：{YYYY-MM-DD}
> 核心特征：{一句话总结}
> 风格稳定性：{高/中/低}

**说明**：{定位和作用}

## 1-N. {各分析维度}
（特征描述 + 使用规律 + 恒定vs适应标注）

## N+1. 人类化特征
**保留特征**：✅ ...
**去除噪音**：❌ ...
**灵活调整**：⚙️ ...
### 去案例化说明
（原始内容 → 抽象化结果）

## N+2. 常见误区
### 误区1：{名称}
- 错误表现 / 正确做法 / 检测方法

*档案分析者：Style Writing Skill*
*完成时间：{YYYY-MM-DD}*
```

### 五个文件核心章节

| 文件 | 命名 | 重点章节 |
|------|------|---------|
| 韵律节奏 | `2022-XX-韵律节奏.md` | 停顿密度统计、节奏模式(A/B/C)、段落vs句子、朗读感、节奏再现指南 |
| 微观语言 | `2022-XX-微观语言.md` | 词汇颗粒度、句式结构、修辞手法、标点节奏、风格禁忌 |
| 宏观结构 | `2022-XX-宏观结构.md` | 框架类型、段落设计、开头策略库(3-6种)、结尾策略库(3-5种)、领域特有 |
| 传播规律 | `2022-XX-传播规律.md` | 钩子设计、情绪曲线、代入感、转发动机、信息差、预期管理、流畅性 |
| 文化认知 | `2022-XX-文化认知.md` | 典故使用、知识领域、术语策略、跨文化、领域特有 |

### 命名与存储

- **文件夹**：`2022-XX-{账号名}_{体裁}_{平台}_风格档案/`
- **文件**：`2022-XX-{维度名}.md`（去掉"层"字）
- **编号**：查看目标文件夹现有最高编号，自动递增
- **体裁**：2字（干货体、观点体、方法体、记录体、故事体等）

### 文件间协调

1. 交叉引用：`参考韵律节奏的{X规则}`
2. 一致性检查：5个文件风格描述无冲突
3. 优先级：韵律 → 语言 → 结构 → 传播 → 文化
