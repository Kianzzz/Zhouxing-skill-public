# AI噪音识别参考

核心原则：避免英文翻译腔、避免AI机械化表达，注重汉语原生表达。仅用于**风格学习模式**。

---

## 三级判断标准

### A级 - 纯AI噪音（直接过滤）

- ❌ AI标志性短语：值得注意的是、综上所述、总的来说、值得一提的是、可喜的是、令人欣慰的是
- ❌ 逻辑永远完美：无犹豫、无修正、无不确定表达
- ❌ 空泛词汇堆砌：智慧、时代、力量、成就等无具体支撑

### B级 - 风格核心（保留标注【风格特征】）

- ✅ 学术风格的正式用语、被动语态
- ✅ 新闻体的结构套路
- ✅ 文学风格的修辞手法

### C级 - 适应性（标注【适应】）

- ⚙️ 书面语程度（根据主题严肃性调整）
- ⚙️ 修辞密度（根据内容类型调整）
- ⚙️ 结构规律性（根据文体调整）

## 检测维度

### 词汇层

| 特征 | 识别方法 |
|------|---------|
| 词汇多样性低 | 段落内高频重复词汇（AI 36.5% vs 人类 58.8%） |
| 倾向正式/学术词 | "释放创新潜力"应说"发挥作用" |
| 倾向大词 | 双音节词占比高（AI 53.2% vs 人类 44.5%） |
| 书面语明显 | 虚词密度高：连词 AI 3.6% vs 人类 1.3% |
| AI高频短语 | "值得注意的是""此外还要"等固定搭配 |

### 句式层

| 特征 | 识别方法 |
|------|---------|
| 英译中句式 | 过度修饰链、并列结构 |
| 结构套路化 | 完全可预测的段落顺序 |
| 转场词过度 | 因果/对比/总结词密度过高 |
| 主语缺乏变化 | "该方法具有…该方法能够…" |
| 并列短语过多 | AI 4.6个/篇 vs 人类 0.8个/篇 |

### 风格层

| 特征 | 识别方法 |
|------|---------|
| 修辞堆砌 | AI修辞是人类5-6倍 |
| 去人格化 | 无第一人称、无个人经历 |
| 直白评价 | 形容词堆砌而非细节呈现 |
| 缺乏口语 | 全是书面正式用语 |
| 逻辑完美生硬 | 无犹豫、无修正 |

## 30秒快速判断

3/5项符合 = 很可能AI生成：

1. 文章结构完全可预测
2. 转场词频繁、位置规则、配搭对称
3. 主语单一重复、句式缺乏变化
4. 无叹号、反问句、口语表达
5. 形容词堆积、直白评价

## 在风格学习中的应用

执行时机：五层分析完成后、生成文件前。

1. 逐层扫描分析结果中是否有AI噪音混入
2. A级删除、B级保留标注、C级标注适应条件
3. 在每层"人类化特征"部分说明过滤理由
