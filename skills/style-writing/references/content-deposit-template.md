# 内容沉淀文件规范

> 仅风格写作/碎片创作模式的沉淀步骤使用。

---

## 用途

风格写作和碎片创作时自动积累的内容规律，帮助在没有完整参考文章的情况下也能创作出符合该类型的文章。

## 文件命名与位置

- 与风格档案同目录：`2022-XX-内容沉淀.md`
- 首次风格写作时自动创建，后续增量追加

## 文件模板

```markdown
# 内容沉淀

> 风格档案：{对应风格名}
> 累计创作次数：{N}
> 最后更新：{YYYY-MM-DD}

## 1. 选题方向库

高频话题、已验证有效的选题模式、选题角度分类。
每条格式：`- 【选题】{方向描述}（来源：第N次创作）`

## 2. 切入视角库

常用切入角度、反常识视角、独特的问题框架。
每条格式：`- 【视角】{视角描述}（来源：第N次创作）`

## 3. 情绪触发库

常触动的情绪类型、痛点清单、共鸣机制。
每条格式：`- 【情绪】{情绪/痛点描述}（来源：第N次创作）`

## 4. 论证模式库

常用论证结构、案例使用方式、说服逻辑。
每条格式：`- 【论证】{模式描述}（来源：第N次创作）`

## 5. 内容密度特征

信息量密度、案例vs观点比例、干货浓度、文章节奏特征。
```

## 沉淀规则

1. 只记录可复用的规律，不记录具体案例内容
2. 与已有条目去重，相似的合并并标注频次
3. 每次更新递增创作次数计数器
4. 保持每个分类 ≤ 20 条（超出时合并低频条目或删除过时条目）
