# 标题创作规则与模板

标题创作的微调手法和重写结构模式。基于肥猫说财（382篇）与反向的猫（765篇）标题规律提炼。

---

## 一、微调版（5个）

保持克制，保留原标题骨架，每个只用1-2种手法。

### 6种微调手法

1. **词汇升级**：普通词 → 高级感词汇（"喜欢"→"执念"、"想法"→"认知"、"状态"→"觉醒"）
2. **增加极端词**：真正、所有、只有、终身、根本、巨大、最
3. **数字具象化**："几个方法"→"三个系统"
4. **调整节奏**：添加标点改呼吸感（"为什么努力没用"→"为什么努力，反而没用"）
5. **括号补充**：添加价值锚点或反转（"坚持做不成（颠覆认知）"）
6. **句式微调**：疑问↔陈述、肯定↔否定

---

## 二、重写版（5个）

提取核心话题词，基于以下8种结构模式重构。**核心话题词不可改变**，必须自然嵌入每个重写版标题中。

### 8种结构模式

**模式A：短句洞察体**（≤15字）
直接抛出观察或观点，极简有力，不修饰不铺垫。句子越短，冲击力越强。
- 太用力的人走不远
- 贫穷久了会变笨
- 审美具有很强的攻击性
- 性格好其实就是认知高
- 密集地做事是很旺人的

**模式B：冒号揭示体**
前半设置框架/领域/概念，冒号（或破折号）后揭示核心洞察。前半制造期待，后半给出意外答案。
- 恐怖的执行力来源于：路径清晰
- 搞钱的顶级心态——"玩"
- 低谷期最大的清醒：贵人不帮弱者，财神不渡怨妇
- 奢侈品思维：不谈功能，只谈感受
- 变强的开始：不自证

**模式C：条件因果体**
"一旦/当...就..." 结构，暗示因果必然性，给读者确定感和行动指引。
- 人一旦开窍，所有事都有规律
- 赚到钱，就会长出新的脸
- 你只要勇敢一次，命运就会识别你
- 一旦主动，你就会感受到命运的推背感
- 体态好了，命就跟着好了

**模式D：对仗张力体**
两个概念对比或对仗，制造认知张力。结构工整，有格言感。
- 弱者慕强，强者慕真
- 强者活在事情里，弱者活在情绪里
- 穷人一直在学习，富人一直在行动
- 高手只相信人性，你却总在赌人品
- 因为急，所以乱，一手好牌打稀烂

**模式E：人称代入体**
"你/你的" 直接与读者对话，制造强代入感。让读者觉得"这说的就是我"。
- 你内核越稳，别人越慌
- 你的情绪，就是你的气运
- 你撤退得越快，你的价值越高
- 你反复经历的事，就是你未完成的功课
- 你怎么对待自己，世界就怎么对待你

**模式F：祈使劝诫体**
"不要/一定要/建议你" 开头的直接劝诫，语气果断，像一个过来人在耳边说话。
- 不要在小事上消耗你的孩子
- 一定要释放自己的强者气息
- 建议你买便宜的衣服
- 千万不要走进一个需要你牺牲，才能正常运转的系统
- 永远记住：先上车再调整姿势

**模式G：疑问好奇体**
提出反常识问题或现象疑问，制造信息缺口。问题本身即观点。
- 为什么越有钱的人，越相信风水？
- 为什么有的人明明长得一般，但就是感觉好看
- 为什么高价值信息从不流向普通人
- 让人着迷的强者气息是怎么来的？
- 为什么说高敏感人往往是大后期人格？

**模式H：权威背书体**
名人/机构/经典 + 冒号 + 洞察观点。借势权威，降低读者抵抗。
- 乔布斯："信噪比"让99%的人注定一事无成
- 哈佛长达85年研究：决定你幸福的，不是钱，也不是婚姻
- 孙子兵法讲透了女人变强的方法
- 毛选：沟通能力强的人，说话顺序都是这样的
- 清华老师：我见过最有出息的孩子，都来自"松弛的家庭"

### 情绪加速器

每个标题可叠加1-2个情绪加速器（不可堆砌过多，保持自然）：

| 加速器 | 说明 | 示例 |
|--------|------|------|
| 极端词 | 真正、所有、顶级、恐怖、致命、疯狂、无一例外 | 真正要起飞的人，无一例外安静的可怕 |
| 反常识 | 颠覆读者默认认知 | 真正的强大，是示弱 |
| 身份标签 | 强者/弱者/穷人/高手/中年人/普通人/高敏感人 | 普通人最大的福报，是早早意识到身后空无一人 |
| 数字锚定 | 用具体数字增加可信度和具象感 | 90%的人都是在假复盘，真正厉害的人，只做这4步 |
| 隐喻升维 | 用高级概念映射日常概念 | 金钱是一块消音棉 |
| 场景具象 | 用具体画面替代抽象描述 | 赚到钱，就会长出新的脸 |

### 选用规则

1. 5个标题覆盖至少4种不同结构模式
2. 标题长度要有分布：至少1个≤12字短句体，至少1个≥20字长句体，其余自由
3. 每个标题叠加1-2个情绪加速器，不可堆砌3个以上
4. 核心话题词必须自然嵌入，不可生硬拼接

---

## 三、质量标准

- 完整句子，读者看得懂且有吸引力
- 严禁加入个人/他人案例
- 微调版：克制，每个只用1-2种手法
- 重写版：结构模式多样，核心话题词必须出现
- 多风格时：微调版共用，重写版可微调用词但核心词不变
- **禁止"不是A，而是B"句式**：此句式已列入全文禁忌（参见SKILL.md创作规范），标题同样适用。用直接陈述正面观点替代
