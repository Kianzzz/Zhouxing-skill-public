# 质量自检清单

> **仅风格学习模式（Mode 1）使用。** 风格写作/碎片创作的审校逻辑已内置于 SKILL.md Task 子任务 + human-writing-guide.md，不再读取本文件。

---

## 风格学习 - 检查清单

### 五文件完整性与一致性

- [ ] 5个文件全部生成（韵律/语言/结构/传播/文化）
- [ ] 5个文件风格描述无冲突
- [ ] 交叉引用正确
- [ ] 韵律节奏和微观语言的节奏描述一致

### 去案例化（最关键）

- [ ] 未出现原文具体案例、人名、品牌名、特定词汇、具体短语
- [ ] 未出现可直接套用的比喻或故事
- [ ] 所有描述已抽象为【变量】形式
- [ ] 所有示例使用【】标记

### 可替换性验证

- [ ] 测试覆盖 ≥ 15个描述
- [ ] 失败率 = 0
- [ ] 所有【恒定】特征已测试

### 韵律节奏专项

- [ ] 完成停顿密度统计
- [ ] 明确判断节奏模式（A/B/C）
- [ ] 区分段落数和句子数
- [ ] 无优化建议，聚焦描述+再现
- [ ] 未混淆视觉分段和真实句长

### 通用

- [ ] 领域特有维度已添加
- [ ] 【恒定】vs【适应】已标注
- [ ] 常见误区章节已添加
- [ ] 人类化特征部分已完成
- [ ] AI噪音已过滤
- [ ] 文件命名格式正确

### 反AI粗糙感维度提取

- [ ] 微观语言层：已提取非功能性口语碎片（碎片类型、插入位置）
- [ ] 宏观结构层：已提取结构不对称模式（长短差异比例）
- [ ] 宏观结构层：已提取功能性冗余模式（冗余类型偏好、频率、位置）
- [ ] 传播规律层：已提取情绪断点模式（断点类型偏好、频率）
