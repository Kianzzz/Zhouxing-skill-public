# 完整示例

## 示例 1：需求理解阶段的提问

**用户输入**：「我想做一个插件，能够快速记录我的灵感。」

**回应**：

好的！我理解你想要一个快速记录灵感的插件。为了设计出最适合你的方案，让我问你几个问题：

**关于记录方式**：
1. 你希望怎么触发记录功能？
   - A) 点击左侧边栏的按钮
   - B) 使用快捷键（如 Ctrl+Shift+N）
   - C) 通过命令面板搜索
   - D) 以上都要

2. 记录灵感时，你希望：
   - A) 弹出一个输入框，输入后自动保存
   - B) 直接在当前笔记插入内容
   - C) 在侧边栏有个常驻的输入区域

**关于存储位置**：
3. 灵感应该保存到哪里？
   - A) 统一保存到一个「灵感收集箱」笔记
   - B) 每条灵感创建一个新笔记
   - C) 保存到今天的日记笔记

4. 灵感内容需要包含哪些信息？
   - 只有文字内容？
   - 需要自动加上时间戳？
   - 需要打标签分类？

**关于优先级**：
5. 这个插件最重要的是什么？
   - 速度（1秒内完成记录）
   - 方便（尽量少点击）
   - 灵活（可以自定义很多选项）

## 示例 2：规划设计阶段的界面展示

**规划文档**：

### 插件规划：灵感速记本

**1. 核心功能（必须实现）**
- 快捷键触发输入框（Ctrl/Cmd + Shift + I）
- 自动保存到「灵感收集箱.md」笔记
- 自动添加时间戳

**2. 重要功能（优先实现）**
- 左侧边栏快捷按钮
- 可在设置中自定义保存位置

**3. 可选功能（后续迭代）**
- 支持添加标签
- 灵感列表查看功能

### 界面设计

**弹窗输入（推荐）**：
```
┌─────────────────────────────────────┐
│  💡 记录灵感                         │
├─────────────────────────────────────┤
│                                     │
│  ┌───────────────────────────────┐ │
│  │  在这里输入你的灵感...         │ │
│  │                               │ │
│  └───────────────────────────────┘ │
│                                     │
│           [💾 保存]  [❌ 取消]      │
└─────────────────────────────────────┘
```

### 使用流程

1. 灵感来临时，按 `Ctrl/Cmd + Shift + I`（或点击左侧💡图标）
2. 弹出输入框，输入灵感内容
3. 点击「保存」或按 Enter
4. 灵感自动保存，格式：`- [2026-01-05 14:30] 你输入的灵感内容`

### 设置页设计

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| 保存位置 | 灵感保存到哪个笔记 | 灵感收集箱.md |
| 时间格式 | 时间戳的显示格式 | YYYY-MM-DD HH:mm |
| 启用快捷键 | 是否启用默认快捷键 | 开启 |

## 示例 3：代码交付时的说明

插件开发完成后的交付格式：

### 插件文件

**1. manifest.json**
```json

  "id": "quick-idea-capture",
  "name": "灵感速记本",
  "version": "1.0.0",
  "minAppVersion": "0.15.0",
  "description": "快速记录灵感的插件",
  "author": "用户名",
  "isDesktopOnly": false
}
```

**2. main.js**
（完整代码...）

### 安装步骤

1. **找到 Obsidian 库文件夹**
   - Mac: `/Users/用户名/Documents/我的Obsidian库`
   - 或在 Obsidian 中：设置 → 文件与链接 → 查看「库文件夹路径」

2. **进入插件目录**
   - 找到 `.obsidian/plugins/` 目录

3. **创建插件文件夹**
   - 命名为：`quick-idea-capture`

4. **复制文件**
   - 将 `manifest.json` 和 `main.js` 保存到文件夹

5. **重启 Obsidian**

6. **启用插件**
   - 设置 → 社区插件 → 找到「灵感速记本」→ 启用

### 使用指南

1. 左侧边栏出现 💡 图标 - 点击即可记录灵感
2. 快捷键 Ctrl/Cmd + Shift + I - 快速打开输入框
3. 设置位置：设置 → 灵感速记本 → 可自定义保存位置
